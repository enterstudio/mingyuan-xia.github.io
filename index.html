Projects

AppAudit: Find Android Apps That Leak Personal Data


Software security
We develop AppAudit, a tool that can analyze an Android app and report if it leaks personal data stored on the smartphone. AppAudit is designed to be used by both end users and academic research. For end users, AppAudit serves as a sheild against dangerous apps. For research, AppAudit can report fine-grained dump about what the app is doing, down to the every specific code location.

AppFence: Sandboxing Apps


Software security
Ever tried the privacy browsing mode of web browers? Now AppFence extends this to all Android apps. AppFence is a universal sandbox for Android apps. Apps that enter AppFence appear like newly installed. When app exits from the AppFence, all traces and data produced are scrubed, like it never happened.
